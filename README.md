## Getting Started

#### See live

Live version: [https://brainhub-task.netlify.app/](https://brainhub-task.netlify.app/)

#### Run locally

1. You need API url to make it work create .env.development file with entry:
   <br>
   `GATSBY_API_URL="xxx"`

2. Install dependency

```bash
   $ npm i
```

3.

```bash
  $ npm run start
```
